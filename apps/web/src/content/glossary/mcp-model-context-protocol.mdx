## Qu'est-ce que le Model Context Protocol ?

Le **Model Context Protocol (MCP)** est un standard open source publié par Anthropic en 2024. Il définit une interface commune — un « port USB-C pour IA » — permettant à n'importe quel LLM de se connecter à des outils, bases de données ou services externes sans écrire d'intégration sur mesure pour chaque combinaison.

### Impact pour les entreprises

MCP supprime la fragmentation des intégrations agent. Avant MCP, connecter un agent à un CRM, un outil de ticketing et un ERP nécessitait trois connecteurs distincts à maintenir. Avec MCP, un seul serveur expose l'ensemble des ressources disponibles.

Claude, Cursor, Windsurf et de nombreux frameworks LLM supportent déjà MCP nativement. Les écosystèmes de serveurs MCP communautaires couvrent GitHub, Notion, Slack, Google Workspace et des dizaines d'autres services.

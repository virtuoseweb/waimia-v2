La plupart des gens utilisent Claude comme un moteur de recherche amélioré. Ils posent une question, ils obtiennent une réponse, ils reformatent à la main. Résultat : 60 % du temps gagné évaporé en retouches.

Cette formation vous apprend à construire des prompts qui produisent exactement la sortie que vous voulez, du premier coup, de façon reproductible — sans écrire une seule ligne de code.

## Le problème que résout cette formation

Un bon prompt, c'est comme une bonne brief créative. Ça n'improvise pas. Ça structure. Ça anticipe les ambiguïtés. Ça définit le format avant de définir le contenu.

Le framework que vous allez apprendre en 90 minutes — 4 blocs, XML, system prompt — est celui qu'Anthropic utilise en interne pour former ses propres ingénieurs. La différence, c'est qu'on l'a réécrit pour des profils non-techniques, avec des exemples métier concrets : rédaction commerciale, analyse concurrentielle, qualification de leads, synthèse de réunions.

## Ce que vous construisez pendant la formation

Le module 4 n'est pas un exercice théorique. Vous partez avec un system prompt réel, calibré sur votre métier spécifique. Vous le testez en direct, vous l'ajustez, vous le sauvegardez dans une bibliothèque personnelle que toute votre équipe peut réutiliser.

Un system prompt bien conçu, c'est un collaborateur virtuel avec une personnalité, un rôle, des contraintes et des objectifs clairs. Vous ne faites plus un seul prompt par tâche — vous faites un prompt par type de tâche, une fois, et vous l'utilisez à l'infini.

## Les 5 erreurs que tout le monde fait

Pendant le module 3, vous réécrirez 5 prompts cassés collectés auprès de vrais utilisateurs PME. Ces exemples ont tous le même défaut structurel : trop vague sur le rôle, trop ambigu sur la tâche, muet sur le format. En les corrigeant, vous intériorisez les règles — bien plus efficacement qu'un cours magistral.

Après 90 minutes, vous aurez un réflexe : avant d'envoyer un prompt, demandez-vous si vous avez défini le rôle, le contexte, la tâche et le format. Si non, prenez 30 secondes pour le faire. Ce réflexe seul vous fera gagner 1 heure par semaine.

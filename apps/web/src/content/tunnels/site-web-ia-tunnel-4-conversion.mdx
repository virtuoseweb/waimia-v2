## Questions courantes

**Pourquoi Astro + React plutôt que Next.js ou WordPress&nbsp;?**

Astro génère du HTML statique pré-rendu indexable immédiatement par Google, ChatGPT, Perplexity et Google AI Overview. Next.js sort souvent plus lourd côté JS. WordPress impose une stack PHP+MySQL coûteuse et plus lente pour les LLM. Notre stack vise 100/100 Lighthouse sans coût marginal.

**Comment fonctionne l&rsquo;agent éditeur de contenu&nbsp;?**

Un agent Claude est branché sur votre repo Git. Vous lui parlez en français — « ajoute un cas client ACME » — il édite les fichiers, ouvre une PR, vous validez. Aucun CMS à apprendre. Le contenu est versionné dans Git avec historique complet.

**Combien coûte le maintien après livraison&nbsp;?**

L&rsquo;hébergement Vercel coûte 0–20&nbsp;€/mois selon le trafic. La routine GEO est un forfait mensuel optionnel&nbsp;: à partir de 490&nbsp;€/mois pour Découverte, 1&nbsp;490&nbsp;€/mois pour Standard. Aucun coût caché.

**Qui possède le site après livraison&nbsp;?**

Vous. On vous transfère le site clé en main&nbsp;: code, contenu, compte d&rsquo;hébergement, variables, documentation et procédure de reprise. Pas de captivité agence, pas de plugin propriétaire obligatoire.

**Êtes-vous compatibles avec mon CRM actuel&nbsp;?**

Oui. Connecteurs natifs HubSpot, Salesforce, Pipedrive. Migration de contenu automatisée depuis WordPress, Webflow, Contentful. Devis exact après audit gratuit.

**Que couvre la garantie 14 jours&nbsp;?**

Pendant 14 jours après mise en ligne, on corrige sans surcoût les régressions de suivi, formulaires, performance et contenu validé. Si un livrable ne correspond pas au cadrage signé, on le reprend avant transfert final.

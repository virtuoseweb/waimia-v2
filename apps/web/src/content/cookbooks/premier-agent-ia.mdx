## Vue d'ensemble

Ce cookbook vous guide pas à pas pour créer un premier agent IA capable de traiter des demandes entrantes et de déclencher des actions automatisées.

## Étape 1 — Définir le périmètre

Choisissez une tâche répétitive, bien délimitée : qualification de leads entrants, résumé d'emails, ou extraction de données structurées.

## Étape 2 — Configurer le modèle

Utilisez Claude via l'API Anthropic ou un outil no-code comme n8n pour connecter votre source de données.

## Étape 3 — Écrire le prompt système

Un bon prompt système définit le rôle, le format de sortie et les limites de l'agent.

## Étape 4 — Tester avec des cas réels

Alimentez l'agent avec 10 exemples réels avant de le mettre en production.

## Étape 5 — Monitorer et améliorer

Suivez le taux d'erreur et ajustez le prompt à chaque itération.

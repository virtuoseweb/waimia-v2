## Résumé

OpenAI a mis à jour les capacités vision de GPT-4o avec une meilleure précision sur les tableaux, les formulaires et les documents multi-colonnes. Les PME B2B disposent maintenant d'un outil robuste pour automatiser l'extraction de données structurées.

## Impact opérationnel

Les cas d'usage immédiats : traitement automatique de factures fournisseurs, qualification de devis entrants, extraction de clauses dans des contrats en PDF.

## Points de vigilance

La précision reste variable sur les documents scannés à faible résolution. Un pipeline de validation humaine est recommandé pour les documents critiques.

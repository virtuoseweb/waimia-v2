Migration vers `SectionsRenderer` testée en prod aujourd'hui sur `/offres/composable-pilot`. Pattern retenu&nbsp;:

```ts
const useComposable = Array.isArray(data.sections) && data.sections.length > 0;
```

Si `sections` est défini dans le frontmatter MDX → SectionsRenderer. Sinon → template traditionnel. Compatibilité backward&nbsp;100&nbsp;%, migration page-par-page, zéro casse.

Leçon&nbsp;: ne pas remplacer, ajouter une branche. Le code applicatif évolue sans dette technique.
